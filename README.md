# mathico v0.4
Мной была реализована библиотека дял создания графического интерфейса с помощью
SFML, а также добавлены требуемые для вычислений вспомогательные и
вероятностно-итерационные функции.

На базе этой библиотеки были реализованы следующие функции:

- Визуализация треугольника Серпинского
- Визуализация папоротника Барнсли
- Построение правильных многоугольников
- Сгенерированные картинки можно будет сохранить в формате '.png' или '.jpg'

Визуализация фракталов задаётся лишь количеством точек N, которые будут отрисованы на экране.
Есть немало статей, где объяснено построение тех или иных фракталов. Я реализовал
[Папоротник Барнсли](https://ru.wikipedia.org/wiki/Папоротник_Барнсли) и
[Треугольник Серпинского](https://ru.wikipedia.org/wiki/Треугольник_Серпинского).

Построение многоугольников на целочисленной решетке, близких к правильным - проект
с майской смены Сириуса 2021 года. Можно легко доказать, что почти все многоугольники
нельзя расположить на целочисленной решетке, а следовательно и нельзя сделать
идеальное растровое изображение правильного многоугольника. Программа делает
построение по ребру с вершинами (0, 0) и (x, y), где вторая вершина вволится через
GUI пользователем.

## Сборка и запуск

Требуется пакетный менеджер Conan, система сборки CMake и утилита Make.

Для сборки и запуска проекта написал готовый makefile скрипт. При сборке подтянутся
требуемые зависимости автоматически.

```bash
make run
```
